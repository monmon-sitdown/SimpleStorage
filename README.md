# SimpleStorage

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

2. Navigate to the project directory:

```bash
cd your-repository
```

3. Install dependencies:

```bash
forge install
```

## Running

1. Compile the contract:

```bash
forge build
```

2. Run tests:

```bash
forge test
```
